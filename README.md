# My dotfiles

## Installation

```bash
git clone 'https://github.com/boltlessengineer/dotfiles.git'

cd dotfiles

# install neovim config
ln -shf ./nvim ~/.config/nvim

# install fish config
ln -s ./fish ~/.config/fish

# install lf config
ln -s ./lf ~/.config/lf
```

### install fish
```bash
brew install fish
fish
fisher update
```
